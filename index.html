Fonctions, arguments et style CSS


function display(argument_elementID) {
    var element = document.getElementById(argument_elementID);
	
    if (element.style.display === 'block') {
        element.style.display = 'none';	
    } else {
        element.style.display = 'block';	
    }
}

Différente opériations peuvent s'appliquer aux tableaux


var newTeamMember = team.push('Fred');

Vous avez aussi :

  • pop('...') - qui supprime le dernier item du tableau
  • shift('...') - qui supprime le premier item du tableau
  • unshift('...') - qui ajoute un item au début du tableau
  • indexOf('valeurRecherchée') - qui récupère la position de la valeur définis en paramètre
  • splice(pos, 1); - supprime "1" item sur une position prédéfinis "pos"
